# Velvet Restaurant Deployment (Phase 1)

🌐 **AWS Static Website Deployment – Phase 1**  
This repository establishes the foundation for the **Velvet Restaurant** deployment pipeline. Phase 1 focuses on designing the core front-end architecture, verifying the static web assets locally, and structuring the codebase to be completely ready for cloud migration and containerization.

---

## 📌 Project Architecture

Before moving into automated cloud deployments (Phase 2), this phase outlines the foundational web components of the **Velvet Restaurant** user interface.

### Tech Stack
* **Markup:** Semantic HTML5
* **Styling:** Custom CSS3 (Responsive layouts for mobile, tablet, and desktop viewports)
* **Interactivity:** Vanilla JavaScript
* **Target Environment:** Standard Web Browsers (Chrome, Safari, Edge)

---

## 📂 Repository Structure

The project assets are organized cleanly to allow effortless mapping into web server directories:
